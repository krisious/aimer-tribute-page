# aimer-tribute-page

A modern static tribute page for **Aimer**, built with Bootstrap and custom CSS.

## Highlights
- Responsive hero section with clear call-to-action buttons
- Sticky translucent navigation with active state on scroll
- Embedded YouTube music video highlights
- Embedded Spotify artist player
- Improved visual hierarchy and typography

## Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- Vanilla JavaScript

## Run locally
Because this is a static site, you can open `index.html` directly or serve it with a local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
